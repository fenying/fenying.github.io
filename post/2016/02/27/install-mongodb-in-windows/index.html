MongoDB 在 Windows 下的手动安装与配置

Category: Database Windows MongoDB
该文章迁移自作者的旧博客站点。
源地址：http://fenying.blog.163.com/blog/static/102055993201612741633702。

本文介绍如何在 Windows 系统中安装 MongoDB 并配置为系统服务。

这里使用 mongodb-win32-x86_64-2008plus-3.2.3.zip ，解压到 C:\DevEnv\MongoDB 下。

在这个目录下新建一个文件，命名为 mongo.conf ，内容修改如下：

dbpath=C:\DevEnv\MongoDB\data # 注意路径
logpath=C:\DevEnv\MongoDB\mongodb.log
logappend=true
journal=true
quiet=true
port=27017

然后启动管理员命令行，运行以下命令：

set MONGO_HOME=C:\DevEnv\MongoDB

mkdir %MONGO_HOME%\data

sc create MongoDB binPath= "%MONGO_HOME%\bin\mongod.exe --service --config=%MONGO_HOME%\mongo.conf" start= demand

net start MongoDB

即可完成 MongoDB 的初始化以及 Windows 服务安装。其中 start= demand 表示服务需要手动启动，如果要系统自动启动，修改为 start= auto 。
