MySQL 5.7.11 在 Windows 下的安装配置

Category: Database Windows MySQL
该文章迁移自作者的旧博客站点。
源地址：http://fenying.blog.163.com/blog/static/10205599320161265624955。

今天下载了一个 mysql-5.7.11-winx64.zip，发现里面没有 data 目录了，看了下以前的笔记整理下安装命令。

这里我把 MySQL 解压在 C:\DevEnv\MySQL。

然后启动管理员命令行，运行以下命令：

set MYSQL_HOME=C:\DevEnv\MySQL
set PATH=%PATH%;%MYSQL_HOME%

cd %MYSQL_HOME%
copy my-default.ini my.ini

cd bin
mysqld --initialize-insecure --basedir=%MYSQL_HOME% --datadir=%MYSQL_HOME%\data --console
mysqld install MySQL --defaults-file="%MYSQL_HOME%\my.ini"


# try start & login
net start MySQL
mysql -uroot -p

即可完成 MySQL 的初始化以及 Windows 服务安装。
