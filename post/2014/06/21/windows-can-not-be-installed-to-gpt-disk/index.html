Windows 8.1 无法安装到GPT硬盘的解决方案

Category: OS windows gpt install
该文章迁移自作者的旧博客站点。
源地址：http://fenying.blog.163.com/blog/static/10205599320145210631739/。

今天安装Windows 8.1 Enterprise With Update En-Us，安装到选择分区时提示 Windows can’t be installed to this disk. This disk is of GPT style.

之前第一次安装Windows 8.1 Pro时，就是因为要求使用GPT分区才允许安装，于是我用DiskGenius把分区表改成GPT。现在倒好，是GPT反而不让安装了。

正在奇怪的时候，重启进入BIOS，发现BOOT选项里除了原来的Windows Boot Management和PS DVD，还多了一个UEFI DVD启动项……

于是立即把UEFI DVD设置为第一启动项，但重启却直接进了系统。只好再进入BIOS把除了UEFI之外的全部启动项取消掉，终于成功安装！

也就是说，其实这个安装盘有两种启动方式，根据需要启动即可～
